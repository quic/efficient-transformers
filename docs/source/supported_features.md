# Supported Features

| Feature | Impact |
|---------|--------|
| [Diffusion Models](https://github.com/quic/efficient-transformers/tree/main/examples/diffusers) | Full support for diffuser-based image generation models like Stable Diffusion, Imagen, Videogen enabling efficient image and video synthesis tasks. |
| [Disaggregated Serving for GPT-OSS](https://github.com/quic/efficient-transformers/tree/main/examples/disagg_serving) | Enabled for GPT-OSS models, allowing for flexible deployment of large language models across different hardware configurations. |
| [ONNX Sub-Functions](https://github.com/quic/efficient-transformers/pull/621) | Feature enabling more efficient model compilation and execution on hardware. |
| [BlockedKV attention in CausalLM](https://github.com/quic/efficient-transformers/pull/618) | Implements a blocked K/V cache layout so attention reads/processes the cache block-by-block, improving long-context decode performance. |
| [Compute Context Length (CCL)](https://github.com/quic/efficient-transformers/blob/main/examples/performance/compute_context_length/README.md) | Optimizes inference by using different context lengths during prefill and decode phases, reducing memory footprint and computation for shorter sequences while maintaining support for longer contexts. Supports both text-only and vision-language models. See [sample script](https://github.com/quic/efficient-transformers/blob/main/examples/performance/compute_context_length/basic_inference.py). |
| Sentence Embedding, Flexible Pooling, Multi-Sequence-Length Compilation | Supports standard/custom pooling with AI 100 acceleration and sentence embedding. Enables efficient sentence embeddings via Efficient-Transformers. Compile with one or multiple `seq_len`; optimal graph auto-selected at runtime. See [sample script](https://github.com/quic/efficient-transformers/blob/main/examples/embeddings/sentence_embeddings.py). |
| [SpD, Multiprojection Heads](https://quic.github.io/efficient-transformers/source/quick_start.html#draft-based-speculative-decoding) | Implemented post-attention hidden size projections to speculate tokens ahead of the base model. See [sample script](https://github.com/quic/efficient-transformers/blob/main/examples/performance/speculative_decoding/multi_projection.py). |
| [QNN Compilation Support](https://github.com/quic/efficient-transformers/pull/374) | Enabled for AutoModel classes QNN compilation capabilities for multi-models, embedding models and causal models. |
| [Disaggregated Serving](https://github.com/quic/efficient-transformers/pull/365) | Supports separate prefill and decode compilation for encoder (vision) and language models. |
| [GGUF Model Execution](https://github.com/quic/efficient-transformers/pull/368) | GGUF model execution (without quantized weights). See [sample script](https://github.com/quic/efficient-transformers/blob/main/examples/text_generation/gguf_models.py). |
| Replication of KV | Enabled FP8 model support on [replicate_kv_heads script](https://github.com/quic/efficient-transformers/tree/main/scripts/replicate_kv_head). |
| [Gradient Checkpointing](https://github.com/quic/efficient-transformers/pull/338) | Supports gradient checkpointing in the finetuning script. |
| [SwiftKV — Snowflake/Llama-3.1-SwiftKV-8B-Instruct](https://huggingface.co/Snowflake/Llama-3.1-SwiftKV-8B-Instruct) | Reduces computational overhead during inference by optimizing key-value pair processing, improving throughput. Supports both [continuous and non-continuous batching](https://github.com/quic/efficient-transformers/pull/367). |
| [Vision Language Model](qeff_autoclasses.md#QEFFAutoModelForImageTextToText) | Provides support for the `AutoModelForImageTextToText` class, enabling advanced vision-language tasks. See [sample script](https://github.com/quic/efficient-transformers/blob/main/examples/image_text_to_text/basic_vlm_inference.py). |
| [Speech Sequence to Sequence Model](qeff_autoclasses.md#QEFFAutoModelForSpeechSeq2Seq) | Provides support for `QEFFAutoModelForSpeechSeq2Seq`, facilitating speech-to-text sequence models. See [sample script](https://github.com/quic/efficient-transformers/blob/main/examples/audio/speech_to_text.py). |
| FP8 Execution | Enables execution with FP8 precision, significantly improving performance and reducing memory usage. |
| Prefill Caching | Enhances inference speed by caching key-value pairs for shared prefixes, reducing redundant computations. |
| On Device Sampling | Enables sampling operations directly on the QAIC device, reducing host-device communication overhead and improving throughput. See [sample script](https://github.com/quic/efficient-transformers/blob/main/examples/on_device_sampling.py). |
| Prompt-Lookup Decoding | Speeds up text generation by using overlapping parts of the input prompt and generated text. See [sample script](https://github.com/quic/efficient-transformers/blob/main/examples/performance/speculative_decoding/prompt_lookup.py). |
| [PEFT LoRA Support](qeff_autoclasses.md#QEffAutoPeftModelForCausalLM) | Enables parameter-efficient fine-tuning using low-rank adaptation, reducing computational and memory requirements. See [sample script](https://github.com/quic/efficient-transformers/blob/main/examples/peft/single_adapter.py). |
| [QNN Support](features_enablement.md#qnn-compilation-via-python-api) | Enables compilation using QNN SDK, making Qeff adaptable for various backends. |
| [Embedding Model Support](qeff_autoclasses.md#QEFFAutoModel) | Facilitates the generation of vector embeddings for retrieval tasks. |
| [Speculative Decoding](features_enablement.md#draft-based-speculative-decoding) | Accelerates text generation using a draft model, reducing latency and improving efficiency. See [sample script](https://github.com/quic/efficient-transformers/blob/main/examples/performance/speculative_decoding/draft_based.py). |
| [Finite LoRAx — Multi-Adapter](qeff_autoclasses.md#QEffAutoLoraModelForCausalLM) | Users can activate multiple LoRA adapters compiled with the base model. At runtime, specify which prompt uses which adapter for mixed adapter batches. See [sample script](https://github.com/quic/efficient-transformers/blob/main/examples/peft/multi_adapter.py). |
| Python and CPP Inferencing API | Provides flexibility for inference with Qeff, enabling integration with various applications. See [sample script](https://github.com/quic/efficient-transformers/blob/main/examples/performance/cpp_execution/text_inference_cpp.py). |
| [Continuous Batching](features_enablement.md#continuous-batching) | Optimizes throughput and latency by dynamically batching requests. |
| AWQ and GPTQ Support | Supports advanced quantization techniques, improving model efficiency and performance on AI 100. |
| Successive Session Requests | An API that yields tokens as they are generated, facilitating seamless integration with various applications. |
| Perplexity Calculation | A script for computing model perplexity for evaluation and comparison. See [sample script](https://github.com/quic/efficient-transformers/blob/main/scripts/perplexity_computation/calculate_perplexity.py). |
| KV Heads Replication Script | A sample script for replicating KV heads for LLaMA-3-8B-Instruct, validating changes, and exporting to ONNX. See [sample script](https://github.com/quic/efficient-transformers/blob/main/scripts/replicate_kv_head/replicate_kv_heads.py). |
